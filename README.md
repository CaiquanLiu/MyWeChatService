# 基于Tomcat+Servlet的个人微信公众号服务后台
个人微信公众号“才权的AI小助手”后台服务

## 博客地址：
<http://www.jianshu.com/u/bf03aa158e75>

## 实现功能：
* 完成微信个人公众号的服务器地址绑定；
* 完成微信个人公众号的消息解析；
* 支持微信的文本输入；
* 支持微信的语言输入；
* 接入Turing机器人接口；
* 完成Log4j的集成；
* 实现基于HtmlParser的web页面爬取和内容分析；
